# ระบุรายการที่ต้องการ หน่วย จำนวน ลบ
📧 เมื่อกด "ส่งคำขอเบิก" ระบบจะส่งแบบฟอร์มนี้ไปยัง VP แผนกของท่านเพื่ออนุมัติ
📱 QR Code สำหรับแบบฟอร์มเบิกวัสดุ
⚠️ ตอนนี้เปิดไฟล์จากเครื่องโดยตรง (file://) — QR Code ที่ได้จะใช้งานได้เฉพาะเครื่องนี้เท่านั้น
กรุณา upload ไฟล์ขึ้น server ก่อน แล้วเปิด URL นั้นเพื่อ generate QR ที่ใช้ได้จริง
สแกน QR เพื่อเข้าสู่แบบฟอร์มเบิกวัสดุ
วิธีใช้งาน:
1. เปิดกล้องโทรศัพท์หรือแอป QR Scanner
2. สแกน QR Code ด้านบน
3. กรอกข้อมูลการเบิกในแบบฟอร์ม
4. กด "ส่งคำขอเบิก" เพื่อส่งให้ VP อนุมัติ
5. รอรับอีเมล์ยืนยันการอนุมัติ
✅ อนุมัติ / ไม่อนุมัติ คำขอเบิกวัสดุ
📋
รอรับคำขอเบิกจากพนักงาน
เมื่อพนักงานส่งคำขอเบิก VP จะได้รับอีเมล์พร้อม Link
กด Link ในอีเมล์เพื่อเข้าสู่หน้านี้และดำเนินการอนุมัติครับ
📊 รายงานสรุปการเบิกวัสดุประจำเดือน
